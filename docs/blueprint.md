# **App Name**: BuiltXStudios

## Core Features:

- Welcoming Homepage: Visually appealing and informative homepage with an overview of BuiltXStudios' services and portfolio.
- Games Section: Dedicated section showcasing available games developed by BuiltXStudios.
- Roblox Games Section: Specific area highlighting games developed within the Roblox platform.
- Merchandise Section: Display of 'Coming Soon' merchandise with engaging visuals and placeholder content.

## Style Guidelines:

- Primary color: Dark, deep gray (#212121) to create a luxurious and sophisticated dark mode base.
- Background color: Near-black (#121212) to ensure contrast and depth in the dark theme.
- Accent color: Metallic silver (#C0C0C0) for highlights, borders, and interactive elements, inspired by Rolls Royce.
- Body and headline font: 'Alegreya', a serif with an elegant and intellectual feel.
- Minimalist design with ample whitespace to emphasize content and enhance visual appeal.
- Sleek, simple icons in metallic silver (#C0C0C0) to maintain consistency with the luxurious theme.
- Subtle, refined animations for transitions and interactions, creating a seamless and engaging user experience.
